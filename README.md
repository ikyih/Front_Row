# Front Row
A recreation of Front Row V1 and V2 for MacOS and iOS in SwiftUI.

FRV1 is for WatchOS.
FRV2 is for iOS and MacOS and will NEVER support Apple Watch.

FRV1 refers to the Tiger V1 recreation. FRV2 refers to the recreation of the version before Front Row was removed and Apple TV G1's Menu

Original V2: https://github.com/zzanehip/Apple-TV-Experiment
